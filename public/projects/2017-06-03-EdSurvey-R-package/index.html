EdSurvey - R package (CRAN)

June 3, 2017

Package Description

The EdSurvey package is a tool to read and analyze functions for education surveys and assessments data from the National Center for Education Statistics, including the National Assessment of Educational Progress (NAEP). Due to the large file sizes and complex formatting of NCES data sets, the package offers user-friendly methods to format and conduct statisitical analyses with a small memory footprint.

For more info:

  • https://cran.r-project.org/web/packages/EdSurvey/index.html
  • http://www.air.org/project/nces-data-r-project-edsurvey
